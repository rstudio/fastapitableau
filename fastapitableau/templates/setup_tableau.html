Configure Tableau to access extensions hosted on your RStudio Connect

If you are using Tableau Server or Tableau Online:

  1. Using an administrative account, login to Tableau Server/Online
  2. Navigate to Settings, then Extensions
  3. Under the heading 'Analytics Extensions', select 'Enable analytics extensions for site'
  4. Create a new connection and select the connection type of 'Analytics Extensions API'
  5. Choose if your RStudio Connect server uses SSL
  6. Enter the information for your RStudio Connect Server:
    7. Host: {{ server_domain }}
    Port: {{ server_port }}
  7. Select 'Sign in with a username and password' and enter the credentials:
    8. Username: rstudio-connect
    Password: <any valid API key from RStudio Connect>
  8. Create / Save changes

If you are using Tableau Desktop:

  1. Navigate to Help, Settings and Performance, Manage Analytics Extension Connection...
  2. Select 'TabPy/External API'
  3. Enter the information for your RStudio Connect Server:
    4. Host: {{ server_domain }}
    Port: {{ server_port }}
  4. If desired, select 'Sign in with a username and password' and enter the credentials:
    5. Username: rstudio-connect
    Password: any valid API key from RStudio Connect
  5. Select whether to Require SSL
  6. Save changes
For more information about setting up analytics extensions in Tableau, please see the Tableau documentation.
